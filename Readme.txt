Project Description:

It is a LAN based Desktop Sharing Application in Windows by which a User can use the whole desktop of other User connected by the LAN.

Steps to follow:

Step1: Install Python2.7, PIL library, Win32 API.
Step2: Connect Computer's with LAN for Communication.
Step3: Keep both (c1.py & images folder) at same location at friend's computer whose destop is to be shared.
Step4: Keep both (c2.py & downloads folder) at same location at your computer who wants to use friends desktop.
Step5: Replace "localhost" with Friends Computer Name(eg: Aki) in file c2.py kept at your side at line no. 138.
Step6: Run the file c1.py placed at friends computer and then run c2.py placed at your computer.
